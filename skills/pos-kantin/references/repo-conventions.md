# Repo Conventions

- Root HTML files stay in the repository root.
- Shared CSS stays in `assets/css`.
- Shared JavaScript stays in `assets/js`.
- Page-specific scripts stay in `assets/js/pages`.
- Reusable UI helpers stay in `assets/js/components`.
- Google Apps Script source stays in `apps-script`.
- Step-by-step docs stay in `docs` and should stay short and numbered.

- Keep placeholder config in source control.
- Keep live config out of source control.
- Keep commit messages short and concrete.
