POS Kantin Monorepo

Frontend ringan, backend Apps Script, dan workflow yang rapi untuk kantin sekolah.

Proyek ini menyiapkan alur dari HTML/CSS/JS ke Netlify, Google Apps Script via CLASP, Google Sheets sebagai database, dan GitHub untuk version control.

Masuk ke aplikasi Buka panduan

Frontend

Multi-page vanilla JS dengan dashboard admin, dashboard petugas, transaksi, users, simpanan, dan laporan.

Backend

Apps Script modular dengan action API tunggal dan response JSON yang konsisten.

Docs

Dokumen kecil bernomor supaya setup tetap ringan dan mudah dilanjutkan.
